### 目的
此时vue3.0 已经发布，源码解析越来越有必要去熟悉一下
先从vue 1.0开始进行一个解析
本课程会制作一个简单的 tiny-vue 版本, 看懂以后在去看vue源代码会很快上手

#### 要实现的功能
```html
<div id="a">
  <input v-model="message" />
  <button v-on:click="increase">Increase</button>
  <p v-text="message"></p>
</div>
<script>
  new Vue({
    el: '#a',
    data: {
      message: 1
    },
    methods: {
      increase(){
        this.message += 1
      }
    }
  })
</script>
```
明确一下我们要实现的功能，这也是vuejs 最核心的两个功能
1. 双向数据绑定，或者叫做数据响应化
2. 指令的支持，以及常用的 v-on v-model v-text 等内建指令

具体点来书，双向数据绑定其实需要实现如下几个部分
1. state
2. observer

directive 会复杂些，需要实现如下几个类
1. Directive
2. Compile
3. Dep
4. Watcher
5. v-text v-on v-model等几个内置的 directive

后面几篇会从如下几个方面来详细讲解
* vuejs 架构
* 实现数据响应化
* 实现 directive
* 实现 Watcher 和 Dep
* 实现几个内置的 directive
